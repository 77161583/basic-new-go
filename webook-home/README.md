# Golang学习页面  Nuxt2.15

## Build Setup

```bash
# 安装所有依赖
$ npm install
# 仅安装生产依赖（部署时安装依赖使用此命令）
$ npm install --production

# serve with hot reload at localhost:3012
$ npm run dev
$ npm run dev:test
$ npm run dev:product

# 构建部署包
$ npm run build
$ npm run build:test
$ npm run build:product

# 启动服务
$ npm run start
$ npm run start:test
$ npm run start:product

# pm2部署
$ pm2 start npm --name "project_name" -- run start
$ pm2 start npm --name "project_name" -- run start:test
$ pm2 start npm --name "project_name" -- run start:product
```
